# Project Description:
This project analyzes Pittsburgh Police arrest data to evaluate neighborhood safety and identify trends that contribute to defining the “best” neighborhood in Pittsburgh through a data-driven approach.

# Descriptions/Links to Datasets
* Dataset Name: Arrest Data
* Source: City of Pittsburgh Open Data Portal

* Description:
The dataset includes records of individuals taken into custody by Pittsburgh Police officers. It focuses primarily on arrests associated with more serious crimes such as felony offenses. Each record contains details such as: Arrest date and time, Offense description and type, Police zone, Demographic information of arrestees
